# Wavecon-Telecom

**# PROJECT OVERVIEW**

This project analyzing the impact of the Wavecon 5G rollout across major Indian cities. 𝗧𝗵𝗲 𝗴𝗼𝗮𝗹 was to evaluate business performance before and after 5G launch, focusing on revenue, KPIs, and plan-wise performance.

**#Purpose**
The purpose of this project was to:

1. Assess the impact of the 5G rollout on revenue and customer adoption.
2. Identify high-performing cities and customer segments.
3. Evaluate plan performance and churn rates.
4. Provide actionable recommendations for leadership in a clear, visual storytelling format.
